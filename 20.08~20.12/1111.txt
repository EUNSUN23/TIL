express: server-side framework. js사용한다. react.js, vue.js등과 같은 언어를 사용하기 때문에
풀스택 개발자 선호하는 스타트업에서 각광받는 framework. 아주 베이직하고 light함. (high-level은 아님)

node.js기반.  

npm 으로 설치한다. npm init(package.json설치), npm install express.


nodemon : 서버를 지속적으로 watch해주는 package. 새로고침할때마다 서버 종료/시작 하지 않아도 됨.

npm i -D nodemon
